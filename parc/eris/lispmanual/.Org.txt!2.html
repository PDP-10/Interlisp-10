		Interlisp Reference Manual Organization

This document contains the current organization of the Interlisp Reference Manual.  This is oganized as a table of contents, except that the major divisions are between files, rather than between sections.  I have tried to organize things so that single files contain single subjects, so this is a more natural division.  The words in square brackets (such as [ChapIntro]) refer to file names:  these files can be found on [phylum]<LispManual>, with the file extension .IM.

Note:  The chapters below are given with particular numbers.  These numbers do not appear in the files anywhere; instead, the IM-to-TEX translation program asks the user to specify a number when a chapter is translated.
