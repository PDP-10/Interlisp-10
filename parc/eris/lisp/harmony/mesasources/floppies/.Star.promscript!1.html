\Partition for Workstation
-- Debug
-- confirm
Comment WARNING - PARTITIONING A SYSTEM DISK DESTROYS ALL CONTENT
Confirm Continue?
Pause 4
Comment SECOND CONFIRMATION REQUIRED
Confirm Continue?

-- do it
Physical RD0 NYN
Online RD0
Specify System 6700 8000 9000 9000 9000
Create RD0 YStar 2
System  normal
User  normal
-- 2 SP between User & normal for default size  
Check RD0
Comment Disk partitioned
