PRESSTOIP (PRESS TO InterPress)


     By Kelly Roach.  Last revised 1-Mar-84.  PRESSTOIP allows the
user to convert PRESS files to INTERPRESS files.  To use,
LOAD <LISP>LIBRARY>PRESSTOIP.DCOM and do
	(PRESS.TO.IP PRESSFILE IPFILE)
where PRESSFILE is the source file name and IPFILE is the destination
file name (can be defaulted).  Comments can be sent to ROACH.PA.
