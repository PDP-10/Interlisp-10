We have just found a bug.  It is not in Fugue.6, but it has been in Carol sysouts and lispcore sysouts for the last month without being detected, so Beau declared it a non-showstopper.  However, you should know about it.'
'
The problem':  On a Dlion, type Control-C.  You get MP 9915, as the machine goes into Raid.  Now, press the big red STOP key.  Total Catatonia': MP changes to 1108, but the mouse stops tracking and you have to reload lisp.  Note that this does not happen if you exit raid by going into Teleraid (press UNDO) and do control-D.  Also, people use STOP all the time to get out of random MP codes, so I think that it is only the deadly combination of control-C and STOP that causes this problem.'
'
Steve Purcell''s 30-second diagnosis':  When they last squeezed the DLion microcode, they took out some code for handling the UNDO and STOP keys.  Perhaps that is where the bug was introduced.'
