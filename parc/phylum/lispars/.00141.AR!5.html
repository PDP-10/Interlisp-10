There is a much easier, less grody, and of course, official, way to make an arbitrary function FOO undoable on typein': define a function /FOO that is an undoable version of FOO, and then add (FOO . /FOO) to the variable LISPXFNS.  Thus define'
'
(/PRINTLEVEL (CARVAL CDRVAL)'
   ((LAMBDA (RESULT)'
	(UNDOSAVE (LIST (FUNCTION /PRINTLEVEL) RESULT))'
	RESULT)'
    (PRINTLEVEL CARVAL CDRVAL].'
'
I put this definition on UNDO and restored the old APRINT.'
