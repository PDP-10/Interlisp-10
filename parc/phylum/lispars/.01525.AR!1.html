'
(1)  Some customers want to generate their own Interpress streams.  Now that Interpress is no longer secret, we can give them this ability.  If any user asks, give them the following info':'
'
(OPENIPSTREAM IPFILE REGION HEADING PREAMBLEFONTS)    [function]'
	This function opens the file IPFILE, and creates an Interpress stream to that file.  The user can pass this stream to functions such as PRINT and DRAWLINE to print text and lines on the press stream.  After the stream is closed (with CLOSEF), it can be sent to an Interpress printer with NSPRINT.'
'
	Warning':  currently, some display operations do not work on Interpress streams, like RELDRAWTO and DRAWCURVE.'
'
	REGION is used to set the page margins within which printing will take place.'
'
	HEADING is the heading to be used on each page.  If this is non-NIL, page numbers will also be printed.'
'
	PREAMBLEFONTS is a list of fonts or font descriptors.  If this is not given, it is still possible to print on the Interpress stream, but this allows certain space and time efficiencies.'
