For the past year, we have been using the DLion TTYPort as an RS232 port -- all the same Lisp functions work as they do on the Dolphin (modulo a few lacunae in the TTYPort implementation, such as an oddity in the DTR signal).   If you don''t have a copy of the Fugue Release and/or the Carol Release notes, I suggest you contack 1100Support.pasa; these contain the specific directons for making up a cable to connect to the TTYPort.'
'
New Lisp "heads" are available in alpha testing now to use the RS232C port.  Much improved performance.'
'
The Dorado has a parallel port, but we''ve never put in the microcode which makes it accessible to the RS232 routines; even at that, you would still need the parallel-to-serial board that is used with the Dolphin.  If you are serious about using this on the Dorado, please make your intentions known.'
'
-- JonL --'
