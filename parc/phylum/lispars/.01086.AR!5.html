Disposition: 9-5-84.  Fixed a long time ago, before the Carol release.  There was a bug in the volume allocation map code which caused the free page count to get out of whack from time to time when allocating pages.  At the same time, I created a workaround': call (DISKFREEPAGES volumeName T), and it will recompute the number of free pages.  -- Tayloe.'
