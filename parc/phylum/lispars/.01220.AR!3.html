If you try to READSYS a sufficiently large sysout file, the READSYS will break with an invalid address error trying to read a portion of the pagemap that it hasn''t mapped yet.  Problem is that READPAGEMAP needs to read the pagemap for ALL of the pagemap pages before it reads the rest of the pagemap.'
'
