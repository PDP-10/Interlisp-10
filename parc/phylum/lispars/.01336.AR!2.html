For you Dolphin and Dorado users': On {Phylum}<LispCore>Next> are new versions of Lisp.run and Lisp.syms.  The main change is that you can now sysin from another partition on the same machine, as well as from a remote Pup server.  Syntax is the same as from within Lisp.  E.g., you can say to the exec'
'
>Lisp {DSK5}FOO.SYSOUT'
'
 or even'
'
>Lisp {DSK3}Lisp.virtualmem'
'
Enterprising users will notice that this latter command effectively gives you "checkpoint" and "rollback" by using the Lisp.virtualmem on a second partition as the checkpoint image.  As with inside Lisp, the partition number requested must be less than or equal to 7.'
'
Fitting this feature in required some substantial changes to the sysin code; let me know if you encounter problems.'
'
	Bill'
