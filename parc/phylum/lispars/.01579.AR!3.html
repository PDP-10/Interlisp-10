2 requests for messages when copying a sysout or other large file to floppies':'
'
1.  Just as the copying starts, how about printing a message saying how many floppies will be required, so the user has some notion of what''s in store for him.'
'
2.  Instead of saying INSERT NEXT FLOPPY between floppies, how about saying INSERT FLOPPY #n, for n = 1, 2, 3, ...
