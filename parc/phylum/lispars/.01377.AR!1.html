Subject: Fix Dorado uCode for BIN to not crash if "unused" flag bits are nonzero

Lisp Version:  3-Jun-84 16':29':06

This is a completely useless behavior, and serves only to prevent the staging of any interesting use for those flag bits.  In particular, proper implementation of BOUT in microcode will require another bit.'

The Dolphin microcode properly ignores the unused bits.
