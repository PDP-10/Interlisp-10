Subject: First-left, Left, and Right margins being set to -85

Lisp Version:  7-May-84 16':18':26

Description: Somewhere in the game, TEdit is setting all the margins in the DEFAULT paragraph looks to be -85.  The effect is that any TEdit started after that will never display any text.  Actually, it''s all being displayed single-character wide off the left edge of the window.

Workaround: INSPECT TEDIT.DEFAULT.FMTSPEC, and change the three margin values to zero.  TEdits which start after that should work right.
