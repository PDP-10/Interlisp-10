Description: If a variable is a list, and an element on the list is a bitmap, VARS doesn''t dump it properly -- the list element is just the printname of the bitmap. Maybe PRETTYPRINTYPEMACROS could be set to know about bitmaps?'


Workaround: The simple workaround is to use the UGLYVARS command instead of the VARS command when dumping data structures that might contain bitmaps.
