Although a reasonable argument could be made that the Organization part of the fully-qualified name ("Xerox") could be omitted, it is not reasonable to omit the Domain.  For a user in Pasadena, {Phylex':} is not an adequate description of the file server Phylex':XSIS North.  The name of a device should not be dependent on the current setting of the Domain default, any more than we should omit the device or directory field from a file''s full name just because you happened to be connected to it at the moment.'
