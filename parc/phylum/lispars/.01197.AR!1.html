Subject: Want to be able to use large memory on Dorados

Lisp Version: 14-May-84 20':07':29

Description: '
Currently, even if your Dorado has 8mb of real memory, Lisp only uses 3mb of it.  This is because there isn''t space in segment zero to put a large real page table (the table that maps from real memory address to virtual).  Fixing this requires the ability to put the real page table somewhere else where there IS space; most of the associated work is in the Bcpl booting process.'
'
Schlumberger is the main affected user.
